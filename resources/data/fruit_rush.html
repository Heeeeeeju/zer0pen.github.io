  1. 개발 동기 및 기대 효과

    2. 남녀노소가 즐길 수 있는 게임을 제작하려던 중 평소에 즐기던 Fruit Ninja를 떠올렸으며, 그 게임의 재미요소인 시원한 타격감을 가진 게임을 만들고 싶다고 생각했습니다. 그러던 중 과일을 살아 움직이게 하면 귀여우면서도 잔인하지 않은 것 같기 때문에 과일을 캐릭터로 사용하게 되었습니다. 과일을 이용하여 좋은 타격감을 만들어 내기 위해 과일을 벨　때의 시원한 소리와 과즙이 터지는 모습을 이용하여 사용자가 더욱 재미있게 즐길 수 있도록 하였습니다.

  3. 게임 소개

    4. 회원가입을 하기위해선 ID와 비밀번호를 적은뒤 회원가입 버튼을 눌러야 합니다.
    로그인을 하기위해선 ID와 비밀번호를 적은뒤 로그인 버튼을 눌러야 합니다.

    로고가 트윈 모션으로 위에서 내려오며, 버튼들이 트윈 모션으로 나타나게 됩니다.
    메인에서는 두가지 게임모드 선택과 랭킹보기, 크레딧보기를 할 수 있습니다.

    (플레이어) 가상패드를 통해 움직이며 돌진을 통해 적을 죽입니다.

    (몬스터1 – 파인애플) 일정시간마다 스폰되며, 점점 스폰 간격이 짧아집니다. 플레이어를 쫓아오며, 플레이어가 공격 범위 안에 들어오면 돌진을 합니다.

    (몬스터2 – 수박) 일정시간마다 스폰되며, 점점 스폰 간격이 짧아집니다. 플레이어를 쫓아오며, 플레이어가 공격 범위 안에 들어올시 플레이어를 향해 씨앗을 뱉습니다.

    (몬스터3 – 두리안) 스테이지를 자유롭게 돌아다니며, 공격을 하지 않습니다. 두리안을 죽일 경우 두리안의 악취에 의해 플레이어는 상태이상( 슬로우 )에 걸리게 됩니다.

    (몬스터4(보스) - 람부탄) 덩치가 매우크고, 플레이어를 쫓아오며, 플레이어가 공격 범위 안에 들어올시 8방향으로 가시를 발사합니다. (!) 한방에 죽지않습니다.

  4. 플레이 방법

    5. (타임어택 모드)
    1) 콤보
    적 하나를 벨 때 마다 콤보가 올라가며, 적을 죽일 때 얻게 되는 점수 = (적이 가진 기본점수 * 콤보) 이기에 콤보를 많이 쌓을수록 점수를 많이 올릴 수 있습니다. 돌진을 했을 때 적을 죽이지 못하면 콤보가 0으로 초기화 됩니다.
    2) 규칙
    화면 중앙 상단에 시간이 표시되며 시간이 다 지나게 되면 게임이 끝나게 됩니다.
    3) 보스
    30초가 지나게 되면 보스(‘람부탄’)이 등장하게 됩니다.
    4) UI
    왼쪽 상단의 숫자는 남은 시간, 중앙 상단의 숫자는 콤보, 오른쪽 상단의 숫자는 점수를 의미합니다.

    (무한모드)
    1) 규칙
    시간이 무제한이란 점을 제외하곤 타임어택 모드와 동일한 방식입니다.
    2) 보스
    보스 (람부탄)는 50킬마다 등장하게 됩니다.
    3) UI
    왼쪽 상단의 숫자는 남은 시간, 중앙 상단의 숫자는 콤보, 오른쪽 상단의 숫자는 점수를 의미합니다.

    (일시정지)
    오른쪽 상단의 일시정지 버튼을 누르면 멈추게 되고 메뉴가 나타나게 됩니다. ‘계속하기’ 버튼을 누르면 게임이 다시 이어지게 되고, ‘메인으로’ 버튼을 누르면 메인으로 돌아가게 됩니다..

    (결과창)
    시간모드일 경우 시간이 다 지나거나 죽으면 결과창이 나타납니다. 무한모드일 경우 죽으면 결과창이 나타납니다. 결과창에는 킬, 점수, 최고 콤보, 자신의 최고점수(서버에 저장된) 가 나타나게 됩니다. ‘메인으로’ 버튼을 누를시 메인으로 이동하게 됩니다.

    (크레딧)
    이 게임을 만든 기획자, 개발자, 디자이너를 소개합니다. 오른쪽 위 X모양 박스를 클릭하면 메인으로 돌아가게 됩니다.

    (랭킹)
    실시간으로 사용자들의 순위를 점수와 함께 보여주게 됩니다. ‘돌아가기’ 버튼을 누를 경우 메인으로 돌아가게 됩니다.

    ( 파티클, 검기 이펙트 )
    적을 벨시 과즙(파티클)이 사방으로 튄다. 또한, 하얀색으로 된 검기 이펙트가 생성된다.

    ( 흔적 )
    적이 죽고나서 과즙이 땅에 뭍어나게되고 일정시간 이후 사라지게 됩니다.
